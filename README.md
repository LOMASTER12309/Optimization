# Optimization
Данный пакет функции (в совокупности с проектом **MyMath**) использовался для написания ВКР по модификациям метода **Нелдера-Мида**. Сырой метод имеет кучу недостатков и ограничений.

Данный же пакет позволяет решать задачи условной, многомерной и многоэкстремальной оптимизации (как поотдельности, так и вместе). В качестве решателей выступают **метод Нелдера-Мида (и его модификации), метод штрафных функций и метод дифференциальной эволюции**.

Кроме того здесь реализован генератор многоэкстремальных функции на основе композиции **случайных положительно определённых квадратичных форм**. Точки локальных экстремумов можно при этом можно задавать как вручную, так и любым другим способом.
Генерация квадратичных форм основана на использовании **случайных положительно определённых матриц**, реализованных в проекте **MyMath**.

Проект нуждается в небольшой доработке, однако уже может использоваться для получения результатов. Также планируется переписать его на C++ для ускорения вычислений.
